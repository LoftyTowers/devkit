# Verification scripts non-requirements (SQL Server contract)

## Scope
Explicit guardrails defining what the DevKit does not mandate for verification scripts.

## Rules
- Verification script naming conventions and organisation are not mandated by authoritative Microsoft guidance; the DevKit MUST NOT impose naming or organisation requirements for verification scripts.

## Prohibited patterns
- Inventing or enforcing verification-script naming or organisation conventions as mandatory requirements without authoritative support.

## Allowed deviations
- None.
